# LensIQ 🔍
### AI-Powered On-Device Product Scanner for Retail & E-Commerce

> Point. Scan. Identify. Instantly.

LensIQ is an Android app that identifies retail products in real time using on-device AI — no internet connection required. Built for e-commerce and retail clients who need fast, private, and accurate product recognition at the shelf level.

## ✨ What It Does

- **Real-time product identification** via live camera feed
- **On-device processing** — no data sent to external servers, works offline
- **Confidence scoring** — shows how certain the model is about each result
- **Related item suggestions** — surfaces similar products based on detected category
- **Instant results** — classification happens in milliseconds using TensorFlow Lite

## 🛠 Tech Stack

| Component | Technology |

## 🚀 Getting Started

```bash
git clone https://github.com/SadraAlio/lensIQ.git
```

Open in Android Studio, sync Gradle, and run on a physical device or emulator with camera support.

**Requirements:**
- Android 8.0+ (API 26+)
- Android Studio Hedgehog or later
- Physical device recommended for camera testing
